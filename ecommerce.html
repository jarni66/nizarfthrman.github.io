E-commerce web scraping

Pada project ini dilakukan pengambilan data produk pada sebuah ecommerce dengan menggunakan web scraping. Data yang dihasilkan dari proses web scrapping ini dapat digunakan untuk menganalisis pasar suatu produk. Web scraping dilakukan dengan menggunakan library BeautifulSoup dan selenium. Produk yang menjadi target pada project ini adalah kaos polos.
