Business Decision Research

Posted on Feb 24, 2022

This project is based on DQLab project module to analyze example of realword problem on business decision research analysis. In this project given data from sport center shop then clean the data with dropping some columns, convert date time, categorize some values. Then visualize data line number of customer each years, average transaction each year, customer churn by product category with pie chart. Last, build model to classified is customer churn or not churn based on average transaction, number of transaction, and year difference between first transaction and last transaction.
